# YumRush

YumRush is a multi-role food delivery application built with Python (Tkinter) and MySQL. It simulates a basic delivery platform with support for customers, drivers, and restaurants.

## Features

### User Roles
- Customer
- Driver
- Restaurant

### Customer
- Browse restaurants through a scrollable interface
- View menus
- Place orders

### Driver
- Update availability status
- View assigned deliveries
- Track delivery status

### Restaurant
- Update restaurant information
- Manage menu items
  - Add items
  - Update items
  - Delete items

## Tech Stack

- Frontend: Tkinter (Python GUI)
- Backend: Python (service layer)
- Database: MySQL
- Environment: Nix with Python virtual environment
